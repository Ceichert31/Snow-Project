# It's Snowing!

_It's Snowing!_ is a Mixed Reality game that blends the real world with snow! Snow builds up around your room, and you can interact with it. 

### Requirements
A Quest 3 or greater is required to play. 

### Setup
1. Install the Android Debug Bridge [ADB](https://developer.android.com/tools/adb) and open a new terminal.
2. Download the latest build and copy the file location to the build
3. Connect the Quest headset to the computer using a USB-C cable
4. Run ```adb install -r "path_to_build\build.apk"```
5. Enable Spatial Permissions
6. Enjoy!

### Troubleshooting
If nothing shows up when playing, confirm you have setup your spatial room data.
